What is a Browser? How do they Work?

What is a browser? How do they work?

What is a browser and examples of web browsers: A browser is a type of software application that allows users to access information and content on the World Wide Web(the internet). The web browser uses a client/server interaction to function. The browser acts as the client, which requests access to information from a server to be displayed by the browser for the user to view[1] [2][3][4].

A web browser can give a user access to different types of information and content on the internet, such as websites, webpages, PDFs, images, videos and files[4][5][6].

A web browser is comprised of several User Interface(UI) components that allow users to use the browser to navigate the internet[1][3]. These include things such as:

• A Search Bar: This is a section to allow the user to enter a URL to access a specific website, or search a specific topic on the internet.

• Tabs: These are rectangular UI elements that allow the user to have multiple websites and webpages open at the same time. The design of a tab is similar to the design of folder tabs used in a physical filing systems. This allows the user to quickly switch and navigate between different tabs to access different information as needed.

• Navigation tools and buttons: These are usually represented by icons such as arrows or boxes that allow the user to move back and forth between different webpages on a website.

There a several commonly used web browsers which include[1][2]:

  • Google Chrome
  • Safari
  • Firefox
  • Microsoft Edge

A simple example of using a web browser would be: Using the Google Chrome web browser to access the website YouTube by searching for the URL: https://www.youtube.com in the search bar.

How do web browsers work[1][2][4][5][6]:

The high-level architecture of a web browser can be broken into the following main components:

• The User Interface: This details the elements in the browser that the user interacts with in order to access the internet. These include some of the previously described elements such as the Search Bar, Tabs, and Navigation Tools.

• The Browser Engine: This controls the interactions and actions that occur between the User Interface and The Rendering Engine.

• The Rendering Engine: This renders the requested information or webpage so that it can be displayed for the user to view on the web browser. The rendering engine can interpret HTML and CSS and convert it into the necessary format that will allow the user to view the website as intended.

• The Networking Component: This handles the internet communication and the internet security of the browser. The networking component will use protocols such as Hypertext Transfer Protocol(HTTP), Hypertext Transfer Protocol Secure(HTTPS) and File Transfer Protocol(FTP), to retrieve URLs and access the requested website or webpage.

• The JavaScript Interpreter: If website makes use of JavaScript code, then this component will interpret and execute that code, and then send it to the Rendering Engine so that it can be processed and displayed to the user.

• The Backend User Interface: This section is used to create basic User Interface elements such as boxes.

• The Data Storage Component: This component creates a database on the local drive of the device on which the browser is installed. The database is then used to manage data such cookies and cache. Cookies are used by the website to retain user data and information such as usernames and passwords.

All of these components function together to allow the user to have a smooth browsing experience when using a web browser to access the internet.
