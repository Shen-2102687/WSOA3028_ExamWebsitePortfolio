Convert Numbers to Words

This is a C++ programme that takes in any interger Num, where 0<= Num <= 10 000, and converts it to its English equivalent phrase.

The programme runs on any IDE that can support C++ and has a console to run the code.

To use the programme, run the file on an IDE as described above and simply enter an integer of choice in the console and then press ENTER. The console will output the correct English phrase.

For Example:
Input: 300
Output: Three Hundred
